홈페이지 제작 주문방법🎺

1. 디자인 선택 / 문의




- 만드시려는 홈페이지의 용도에 맞게 디자인을 선택해주세요!

- 이후 인스타그램 DM이나 카카오톡 오픈채팅으로 어떤 홈페이지를 만들고 싶으신지 용도는 무엇인지 문의주세요.
카카오톡 오픈채팅은 이곳 을 누르시면 되고, 인스타그램 아이디는 @let_me_make_your_hompage 입니다.
(기본적인 예의는 갖춰주세요 🙏🙏🙏🙏 같은말 하지 않습니다 ㅎㅎ.
심야,새벽 상관없이 24시간 언제든 문의 주세요~!)

- 꼭 용도와 템플릿이 일치할 필요는 없으며, 고객님께서 원하시는 용도에 맞게 무엇이든 만들어드릴 수 있습니다.
(ex. 개인홈페이지 스타일로 이력서를 만들어 주세요! / 기념일사이트에 구글맵 지도를 추가해주세요! → 모두 가능합니다.)
예를들어, Link ← 와 같이 기념일사이트 Style로 온라인 초대장을 만들 수 있습니다.

- 만드시려는 홈페이지에 어떤 사진을 넣을지, 어떤 내용을 담을지 DM으로 말씀해주세요!

2. 주문/결제


- PC와 스마트폰은 화면의 해상도가 다릅니다. 때문에, PC에선 잘 보이는 홈페이지가 스마트폰에선 깨져서 보이는 경우가 있습니다.

- 반응형 사이트는 해상도에 따라 자동으로 화면조정을 해주는 사이트를 말합니다. 기존 사이트보다 만들기가 까다롭지만,
저는 모든 사이트를 반응형으로 제작하여 드립니다!

- DM / Email로 요구사항을 말씀해 주시면, 사이트 제작에 들어갑니다.

- No show 방지를 위하여. 사이트 제작전, 계약금의 50%를 입금해주셔야 됩니다.

3. 시안/수정


- 2일 이내 사이트를 제작하여 드립니다.
완성된 사이트를 직접 보신 후, 고객님의 요구에 맞추어 추가적인 수정을 무료로 해드립니다.
zoom이나, google 회의를 이용하여, 실시간으로 고객님의 요구에 맞추어 변경해드립니다.
ex) 생각해보니, 배경화면은 이 사진으로 바꾸고 싶어요. / 홈페이지에 적힌 내용을 바꾸고 싶어요

- 최종 수정이 완료되면, 사이트를 구축하여 드립니다.

4. 홈페이지 업로드


- 홈페이지 주소의 경우, 기존에 존재하는 사이트 주소는 사용이 불가능합니다.
(예를들어 www.naver.com이라는 주소를 가진 개인 사이트를 만들 수는 없습니다.)

- 홈페이지 주소는 연간 단위의 유료도메인사이트 / 3개월간 무료, 이후 유료 주소 / 영구적 무료 주소가 있습니다.



1. 유로도메인 사이트의 경우

상업목적의 사이트를 제작하시려는 분들께 추천드립니다.
유료 주소의 경우, 유료도메인사이트 에서 원하시는 사이트 주소를 찾으실 수 있으며 추가금이 소요됩니다.
www.원하는 사이트명.com 과 같은 사이트를 만들 수 있습니다.

2. 1년간 무료, 이후 유료 주소의 경우

단기적으로 사이트를 소유하시려는 분들께 추천드립니다.
저의 경우, 이 방법을 이용하여 사이트를 만들었습니다.
사이트주소가 .com이 아닌 tk / ml / ga / cf로 끝납니다.
(예를들어 www.원하는사이트명.com이 아닌 / www.원하는사이트명.tk가 만들어집니다.)

3. 영구적 무료 주소의 경우

개인 기록용 사이트, 관리 없이 평생 사이트를 소장하고 싶으신 분께 추천드립니다.
무료 주소의 경우, 깃허브 에 가입하셔야 하며, 가입하신 고객님의 유저아이디.github.io 라는 홈페이지가 제작됩니다.
github는 마이크로소프트에서 무료로 제공하는 서비스입니다. 따라서, 보안 / 비용의 걱정을 하지 않으셔도 됩니다.
