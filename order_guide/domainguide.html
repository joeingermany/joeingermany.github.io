홈페이지 도메인 세부사항 안내

1. 유로도메인 사이트의 경우

상업목적의 사이트를 제작하시려는 분들께 추천드립니다.
유료 주소의 경우, 유료도메인사이트 에서 원하시는 사이트 주소를 찾으실 수 있으며 추가금이 소요됩니다.

1. 링크된 주소에 들어가셔서 만드시려는 사이트의 주소를 검색해주세요.
(예를들어 만드시려는 사이트가 www.hompagemaker.com일 경우 .com을 제외한 'hompagemaker'만을 검색해 주세요)

2. 아래의 가격표에서 표시되는 금액은 1년간 금액이며, 1년이 지난 후엔 해당 사이트에 추가 요금을 부과하셔야 됩니다.
따라서, 상업목적의 홈페이지를 만드시려는 분들께 추천드립니다.

2. 1년간 무료, 이후 유료 주소의 경우

단기적으로 사이트를 소유하시려는 분들께 추천드립니다. 저의 경우, 이 방법을 이용하여 사이트를 만들었습니다.

- freenom.com 에서 원하시는 사이트 주소를 찾으실 수 있으며,
1년 후엔 동일한 주소를 사용하기 위해 추과금을 freenom.com에 지불하여야 합니다.

- 사이트의 주소 끝은 .com이 아닌 .tk / .ml / .ga / .cf 로 끝나게 됩니다.
(예를들면 www.naver.tk / www.naver.ml과 같은 사이트가 만들어집니다.)

1. 링크된 주소에 들어가서 만드시려는 사이트의 주소를 검색해주세요.
(예를들어 만드시려는 사이트가 qwee.tk일 경우 'qwee'만을 검색해 주세요)




2. 결과페이지에서, 만드시려는 도메인이 사용 가능한지, 불가능한지 확인하실 수 있습니다.
(qwee의 경우 qwee.tk / qwee.ml / qwee.cf의 경우 이미 사용중이므로 qwee.ga만 만들 수 있는 상태입니다.)

3. 영구적 무료 주소의 경우

개인 기록용 사이트, 관리 없이 평생 사이트를 소장하고 싶으신 분께 추천드립니다.

무료 주소의 경우, 깃허브 에 가입하셔야 하며, 가입하신 고객님의 유저아이디.github.io 라는 홈페이지가 제작됩니다.

1. 우측 상단의 'Sign up' 버튼을 눌러주세요



2. 이메일과 비밀번호, username을 작성해주세요. 여기서 username은 사이트 도메인에 사용됩니다.
예를들어 hompagemaker.github.io 라는 사이트를 만들고 싶을 경우,
username에 github.io를 제외한 hompagemaker 만 적은 상태로 가입해야 됩니다.
